# Project Roadmap

> 締め切り: 3時間スプリント
> 最終更新: 2026-02-21

---

## Phase 1: MVP（3時間スプリント）

### Epic 1: ゲーム基盤
- [ ] YouTube動画埋め込み再生（IFrame Player API）
- [ ] YouTube字幕APIによる歌詞データ自動取得
- [ ] 字幕パース + ローマ字変換

### Epic 2: コアゲームプレイ
- [ ] タイピング判定エンジン（ローマ字照合、スペース圧縮）
- [ ] MediaPipeハンドジェスチャー検出（手上げ判定）
- [ ] Hypeゲージ（自然減少 + 手上げ回復）
- [ ] スコア計算（単語 × Hype倍率）

### Epic 3: UI/画面
- [ ] `/play` - URL入力画面
- [ ] `/play/[id]` - ゲームプレイ画面（動画 + 歌詞 + タイピング + Hype）
- [ ] ゲーム結果画面

```api
GET  /play                    → URL入力ページ
GET  /play/[videoId]          → ゲームプレイページ
GET  /api/captions?v={videoId} → 字幕データ取得API
```

> **並列開発可能:** Epic 1（字幕取得・パース）と Epic 2（タイピング判定・ジェスチャー検出）は**共通のデータ型（歌詞配列）を定義すれば並列で実装可能**。片方がモックデータで開発を進められる。
