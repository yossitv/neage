# User Stories

## As a プレイヤー

### Story 1: YouTube URLでゲームを開始する
**As a** プレイヤー
**I want** YouTube動画のURLを入力してゲームを始めたい
**So that** 好きな曲で歌詞タイピングゲームができる

**Acceptance Criteria:**
- [ ] `/play` ページでYouTube URLを入力できる
- [ ] URL入力後、`/play/[videoId]` に遷移する
- [ ] 字幕データが自動取得される
- [ ] 無効なURLの場合エラーメッセージが表示される
- [ ] 字幕がない動画の場合エラーメッセージが表示される

---

### Story 2: 歌詞をタイピングしてスコアを稼ぐ
**As a** プレイヤー
**I want** 動画に合わせて表示される歌詞をタイピングしたい
**So that** 歌詞を覚えながらタイピング練習ができる

**Acceptance Criteria:**
- [ ] YouTube動画が再生され、再生時間に連動して歌詞が表示される
- [ ] 歌詞のローマ字表記が表示される
- [ ] キー入力がリアルタイムで判定される（正誤表示）
- [ ] 正しくタイピングするとスコアが加算される

---

### Story 3: 手を上げてHypeを管理する
**As a** プレイヤー
**I want** 手を上げるジェスチャーで難易度を調整したい
**So that** 速すぎる曲でも楽しくプレイできる

**Acceptance Criteria:**
- [ ] Webカメラで手のジェスチャーが検出される
- [ ] 手を上げると現在の単語がpassされる
- [ ] 手を上げるとHypeゲージが回復する
- [ ] Hypeゲージは時間経過で自然減少する
- [ ] 次の歌詞が来たら手上げ判定がリセットされる

---

### Story 4: ゲーム結果を確認する
**As a** プレイヤー
**I want** ゲーム終了後にスコアと成績を確認したい
**So that** 自分のパフォーマンスがわかる

**Acceptance Criteria:**
- [ ] 動画終了後に結果画面が表示される
- [ ] 最終スコア、タイピング精度が表示される
- [ ] 「もう一度プレイ」ボタンで同じ曲を再プレイできる
- [ ] 「別の曲を選ぶ」ボタンで `/play` に戻れる
